# greentext-extra
it's like greentext but with more features
# formatting
`- text`
colour is orange
replaces the dash with a bullet point
`? text`
colour is blue
underlines text
`---`
just a hr
`! text`
colour is yellow
monospace text
`^ text`
colour is purple
bold text
`| text`
colour is gray
italic text
`> text`
make it green

you could remove the space between the syombol and the text for all of the above

`text`
regular text
